# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/7cae4608-454d-4a5f-99d9-fb9c8d8c8a26

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env](.env) to your Gemini API key (copy `.env.example` as a starting point)
3. Run the app:
   `npm run dev`
